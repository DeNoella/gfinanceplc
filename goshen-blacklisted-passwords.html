02 ‚Äî Why This Matters

The Problem With "Smart-Looking" Weak Passwords

Many employees use institution-related passwords believing they are complex. Here is why this reasoning fails.

01
Password Meters Lie
Strength meters measure entropy ‚Äî length, character variety ‚Äî not guessability. A password like Goshen@2050! scores "Very Strong" but is in every attacker's Goshen-specific wordlist.
02
Targeted Wordlist Attacks
When hackers target a specific institution, they build custom dictionaries using the company name, location, founding year, and branch names. This is standard attack methodology for financial targets.
03
Leet-Speak Does Not Help
Replacing letters with symbols ‚Äî G0sh3n! or @men123 ‚Äî is well known to attackers. Modern cracking tools apply these substitutions automatically within milliseconds.

08 ‚Äî What To Do Instead

Creating Passwords That Are Actually Strong

üé≤
Three Random Words
The NCSC recommends combining three random, unrelated words into a passphrase. Long, memorable, and nearly impossible to guess.
correct-horse-battery ‚Üí CorrectHorseBattery9!
üîë
Use a Password Manager
Bitwarden, 1Password, or KeePass generate and store truly random passwords so you never need to memorise them.

üìè
Length Over Complexity
A 16-character phrase of simple words is mathematically stronger than an 8-character jumble. Length is the primary driver of security.
MangoBridgePurpleCloud2025! ‚úì
üîÑ
Unique Per System
Never reuse passwords. If one system is breached, attackers immediately try the same credentials everywhere else ‚Äî called credential stuffing.
Different password for email, core banking, VPN, HR
üì±
Enable Two-Factor Auth (2FA)
Even if your password is somehow compromised, 2FA prevents access without your physical device. Enable it on all accounts that support it.
Password + OTP app = two locks on the door
üìÖ
Rotate Every 90 Days
Goshen Finance policy requires password changes every 90 days. Use this as an opportunity to generate a completely new, unrelated password.
Each rotation = entirely new random phrase
